Start by copying any JAR files that the project references to the following directory
C:\Cisco\CallStudio\eclipse\plugins\com.audiumcorp.studio.elements.core\lib
 
Once the JAR files have been copied over you can import add the class file to your CVP call studio Project. Use the following location “deploy/java/application/classes” 

Now close and reopen the project. Once this is done you should now see the application under the elements window. Under Local Elements -> Custom Elements 

The following parameters are currently supported 
To Email
From Email
SMTP Server
SMTP Port
Subject Line
Email Body
Server Connect Timeout
Server Request Timeout
